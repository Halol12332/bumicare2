**BumiCare** is a mobile application designed to track, manage, and visualize key environmental metrics. It enables users to analyze sustainability trends and makes complex environmental data accessible and actionable on the go.

## ✨ Key Features

* **Mobile-First Experience:** Designed specifically for Android (and cross-platform compatible) to provide sustainability tracking at your fingertips.
* **Interactive Dashboards:** Real-time data visualization to track metrics like carbon footprint and energy consumption.
* **Data-Driven Insights:** Seamlessly fetches and processes environmental data to help users make informed decisions.
* **Scalable Architecture:** Built with Flutter for smooth performance and a premium UI experience.

## 🛠️ Tech Stack

* **Framework:** [Flutter](https://flutter.dev/)
* **Language:** [Dart](https://dart.dev/)
* **IDE:** Android Studio / VS Code

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Ensure you have the following installed and configured on your local machine:

* [Flutter SDK](https://docs.flutter.dev/get-started/install)
* [Android Studio](https://developer.android.com/studio) (with Flutter and Dart plugins installed) or VS Code
* An Android Emulator (configured via Android Studio) or a physical device connected with USB Debugging enabled.

### Installation & Execution

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Halol12332/bumicare2.git](https://github.com/Halol12332/bumicare2.git)
   cd bumicare2
   ```
Install dependencies:
Fetch all required Dart packages by running:

```Bash
flutter pub get
```
Run the application:
Make sure your emulator is running or your physical device is connected, then execute:

```Bash
flutter run
```
(Alternatively, you can open the project folder in Android Studio and click the green "Run" button at the top).
